# sorter5

**Фильтратор повторяющихся вопросов**\n
Позволяет отфильтровать повторяющиеся вопросы в файлах ассистентского формата 
?Вопрос
+Правильный ответ
-Неправильный ответ 1
-Неправильный ответ 2

**Особенности:**
- Графический интерфейс на Tkinter
- Драг-н-дроп через tkinterdnd2
- Настраиваемый коэффициент сходства вопросов*
- Ручная проверка и восстановление вопросов


**Использование:**
1. Загрузите файл через drag-and-drop или диалог выбора
2. Установите коэффициент сходства* (70% по умолчанию)
3. Нажмите "Запустить фильтрацию". Программа сравнивает не только текст самого вопроса, но и все варианты ответа
4. Чтобы удостовериться в корректности фильтрации, можно вручную просмотреть все повторы, которые программа таковыми посчитала, и в случае нахождения ошибки - вернуть вопрос обратно в базу вопросов для дальнейшего экспорта в файл
5. Экспортируйте результат в файл
6. Радуйтесь

**Коэффициент сходства:**\n
Принимает значение от 0 до 100% 
Определяет, насколько вопросы должны быть похожи, чтобы считаться дубликатами. Чем выше процент, тем строже сравнение
  
**Удачи!**
