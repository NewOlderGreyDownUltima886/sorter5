# sorter5

Позволяет отфильтровать повторяющиеся вопросы в файлах формата ассистент
?Вопрос
+Правильный ответ
-Неправильный ответ 1
-Неправильный ответ 2

Базирована на Tkinter, отдельно использована tkinterdnd2 для драг-н-дропа

Для использования нужно загрузить желвемый файл для сортировки, указать коэффициент сходства* (70 по умолчанию) и нажать соответствующую кнопку. Программа сравнивает не только текст самого вопроса, но и все варианты ответа. При любом различии вопросы будут считаться разными
Чтобы удостовериться в корректности фильтрации, можно вручную просмотреть все повторы, которые программа таковыми посчитала, и в случае нахождения ошибки - вернуть вопрос обратно в базу вопросов для дальнейшего экспорта в файл

* Коэффициент сходства (0-100%) определяет, насколько вопросы должны быть похожи, чтобы считаться дубликатами. Чем выше процент, тем строже сравнение.
  
Удачи!
